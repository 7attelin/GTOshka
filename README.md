# Бот-помощник для сдачи нормативов студентам ХИИК СибГУТИ

> Telegram-бот для помощи студентам в подготовке и сдаче вузовских нормативов и нормативов ГТО.  
> Разработан на **Python 3.12+** с использованием библиотеки `python-telegram-bot`.

---

## 🚀 ВОЗМОЖНОСТИ

### 🎓 Вузовские нормативы
- **Нормативы СПО**: Отправка фотографии с нормативами.
- **Нормативы ВО**: Отправка PDF-файла с полным списком нормативов, описание содержимого и пояснения для студентов разных медицинских групп.

### 🏅 Нормативы ГТО
- Выбор возрастной ступени (`6`, `7`, `8`) с указанием возраста.
- Просмотр фотографии нормативов для выбранной ступени.
- Подробные рекомендации по недельной двигательной активности для каждой ступени.
- Прямые ссылки на официальные страницы ступеней на сайте ГТО.

### ℹ️ Инфо
- Ссылка на официальный сайт ГТО.
- Отправка таблицы (Excel) со ссылками на видео-уроки по технике выполнения упражнений (легкая атлетика, волейбол, баскетбол и др.).
- Умное обновление меню после отправки файла, чтобы избежать спама.

## ⚡ БЫСТРЫЙ СТАРТ

### 1. Установка зависимостей
Убедитесь, что у вас установлен **Python 3.12** или выше.

```bash
pip install -r requirements.txt
```

### 2. Получение токена бота
1. Откройте Telegram и найдите [@BotFather](https://t.me/BotFather)
2. Отправьте команду `/newbot`
3. Придумайте имя и username для бота (username должен заканчиваться на `bot`)
4. Скопируйте полученный токен.

`P.s - Бот уже создан под ником @GtoshkaBot для просмотра можете распаковать zip папку с целым проектом, которую я вам прислал на почту и активировать бота запустив основной файл bot.py
Предварительно установив все библиотеки из requirements.txt`

### 3. Настройка
Создайте файл `.env` в корне проекта:

```bash
cp .env.example .env
```

Откройте `.env` и вставьте ваш токен:

```env
TOKEN=ваш_токен_от_botfather
```

### 4. Подготовка файлов
Бот ожидает наличие определенных папок и файлов для корректной работы.

#### 📂 Структура папок
Все медиафайлы должны находиться в папке `data` относительно корня проекта.

- **Изображения**: `data/images/`  
  Сюда помещаются фотографии нормативов (`.jpg`).  
  Имена файлов должны соответствовать настройкам в `config.py` (по умолчанию: `spo_1.jpg`, `gto_stage_6.jpg`, `gto_stage_7.jpg`, `gto_stage_8.jpg`).

- **Документы**: `data/files/`  
  Сюда помещаются PDF-файлы и таблицы.  
  По умолчанию бот ищет:
  - `Контрольные и практические задания для ВО.pdf` (нормативы ВО)
  - `video_links_table.xlsx` (таблица с видео-уроками для раздела "Инфо")

> ⚠️ **ВАЖНО**: Если вы используете другие имена файлов, обязательно обновите словари `IMAGES` и `FILES` в файле `config.py`.

### 5. Запуск бота
```bash
python bot.py
```
При успешном запуске вы увидите сообщение:  
`"Бот успешно запущен. Ожидание команд."`

## ⚙️ НАСТРОЙКА

Все тексты, ссылки и пути к файлам вынесены в файл `config.py` для удобного редактирования без изменения логики бота.

### Основные параметры
| Параметр | Описание |
|----------|----------|
| `INSTITUTE_SITE` | Ссылка на сайт вашего института |
| `GTO_MAIN_SITE` | Основная ссылка на сайт ГТО |
| `GTO_STAGE_LINKS` | Словарь со ссылками на конкретные ступени ГТО |
| `GTO_RECOMMENDATIONS` | Подробные тексты рекомендаций для каждой ступени (поддерживает Markdown) |

### Обновление путей к файлам
Если вы изменили имена файлов в папках `data/images` или `data/files`, отредактируйте соответствующие словари в `config.py`:

```python
IMAGES = {
    "spo_1": "data/images/spo_1.jpg",
    "gto_stage_6": "data/images/gto_stage_6.jpg",
    # ...
}

FILES = {
    "vo_norms_pdf": "data/files/Контрольные и практические задания для ВО.pdf",
    "video_links_table": "data/files/Контрольные нормативы и упражнения.xlsx",
}
```

## 🧭 ОСОБЕННОСТИ НАВИГАЦИИ

Бот использует максимально простой подход к взаимодействию для удобства пользователя:

1. **Меню**: При навигации по меню сообщения **РЕДАКТИРУЮТСЯ** (обновляется текст и клавиатура), чтобы не засорять чат.
2. **Контент (Фото/Файлы)**: При запросе нормативов старое сообщение меню **УДАЛЯЕТСЯ**, и бот отправляет новые сообщения с файлами/фотографиями.
3. **Защита от спама**: В разделе "Инфо" при запросе таблицы с видео сообщение меню **ОБНОВЛЯЕТСЯ** (статус кнопки меняется), а файл повторно не отправляется, если он уже был выслан в текущей сессии взаимодействия.
